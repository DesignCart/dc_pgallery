<h1>Design Cart Gallery for PrestaShop (dc_pgallery)</h1>
<p>Design Cart Gallery is a free PrestaShop module that solves one of the most common limitations of content management inside online stores: the inability to easily place professional image galleries exactly where you need them.</p>
<p>Instead of manually creating HTML layouts, installing page builders, or managing complex media systems, you simply upload images to a folder on your server and insert a shortcode into your content.</p>
<p>The module automatically scans the folder, generates the gallery, loads only the required assets, and displays a responsive gallery directly inside product descriptions, CMS pages, CMS categories, and other supported HTML content fields.</p>
<h2>Why was this module created?</h2>
<p>Many PrestaShop stores need more than the standard product gallery.</p>
<p>Store owners often want to:</p>
<ul>
<li>
<p>show completed projects,</p>
</li>
<li>
<p>present customer installations,</p>
</li>
<li>
<p>create visual portfolios,</p>
</li>
<li>
<p>display inspiration galleries,</p>
</li>
<li>
<p>build image-rich landing pages,</p>
</li>
<li>
<p>create reference sections,</p>
</li>
<li>
<p>showcase before-and-after photos.</p>
</li>
</ul>
<p>Unfortunately, standard PrestaShop functionality makes these tasks unnecessarily difficult.</p>
<p>Design Cart Gallery solves this problem with a shortcode-based system that can be inserted almost anywhere.</p>
<h2>Main Features</h2>
<h3>Shortcode-Based Galleries</h3>
<p>Create galleries using a simple shortcode:</p>
<pre><code>{dcgallery source="portfolio" mode="normal" columns="4"}
</code></pre>
<p>No HTML coding required.</p>
<h3>Server Folder Integration</h3>
<p>Images are loaded directly from folders stored on your server.</p>
<p>Example:</p>
<pre><code>img/mygalleries/portfolio/
</code></pre>
<p>Simply upload new images and the gallery updates automatically.</p>
<h3>7 Gallery Modes</h3>
<p>The module supports seven different display modes:</p>
<ul>
<li>
<p>Normal Grid</p>
</li>
<li>
<p>Tiles (Masonry)</p>
</li>
<li>
<p>Slideshow</p>
</li>
<li>
<p>Carousel</p>
</li>
<li>
<p>Coverflow</p>
</li>
<li>
<p>Cards</p>
</li>
<li>
<p>Thumbnails Gallery</p>
</li>
</ul>
<p>Each mode is designed for different use cases and visual styles.</p>
<h3>Responsive Design</h3>
<p>All gallery layouts work on desktop, tablet, and mobile devices.</p>
<h3>Lightbox Support</h3>
<p>Integrated GLightbox support provides:</p>
<ul>
<li>
<p>image zoom,</p>
</li>
<li>
<p>fullscreen viewing,</p>
</li>
<li>
<p>gallery navigation,</p>
</li>
<li>
<p>touch support.</p>
</li>
</ul>
<h3>Multiple Galleries Per Page</h3>
<p>Place several galleries on the same page without conflicts.</p>
<p>Each gallery instance is generated independently.</p>
<h3>Lazy Asset Loading</h3>
<p>The module automatically loads only the CSS and JavaScript required by the selected gallery mode.</p>
<p>This helps reduce unnecessary page weight and improves performance.</p>
<h2>Supported Locations</h2>
<p>The shortcode can be used in:</p>
<ul>
<li>
<p>CMS pages,</p>
</li>
<li>
<p>CMS category descriptions,</p>
</li>
<li>
<p>product descriptions,</p>
</li>
<li>
<p>short product descriptions,</p>
</li>
<li>
<p>other HTML fields processed by PrestaShop content filters.</p>
</li>
</ul>
<h2>Gallery Modes Overview</h2>
<h3>Normal Grid</h3>
<p>Perfect for portfolios, product examples, and image collections.</p>
<h3>Tiles (Masonry)</h3>
<p>Pinterest-style layout for images with different proportions.</p>
<h3>Slideshow</h3>
<p>Classic single-slide presentation.</p>
<h3>Carousel</h3>
<p>Multiple visible slides with horizontal navigation.</p>
<h3>Coverflow</h3>
<p>Modern 3D presentation effect.</p>
<h3>Cards</h3>
<p>Interactive card-stack animation.</p>
<h3>Thumbnails Gallery</h3>
<p>Large image with thumbnail navigation underneath.</p>

<p>Design Cart Gallery includes several security measures:</p>
<ul>
<li>
<p>path traversal protection,</p>
</li>
<li>
<p>secure folder validation,</p>
</li>
<li>
<p>HTML escaping,</p>
</li>
<li>
<p>compatibility with PrestaShop HTMLPurifier,</p>
</li>
<li>

<p>GLightbox</p>
</li>
<li>
<p>Macy.js</p>
</li>
<li>
